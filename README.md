# Installatain

**Download using Terminal**
1. git clone: https://github.com/syskraken/krakenprime.git
2. cd krakenprime
3. ./setup.bat
4. ./build.bat
5. cd dist
6. ./KrakenPrime.exe

**Download as zip folder**
1. Download zip folder
2. Extract the zip folder
3. Execute setup.bat
4. Execute build.bat
5. Go to dist folder
5. Execute KrakenPrime.exe

**LDPlayer Configration**
You need to configure LDplayer emulator settings to make it work properly
1. In the settings go to display tab and set the resolution to 1600x900 (240 DPI)
2. In the Others tab set the ADB Debugging to enable local connection

for more info watch youtube tutorial: https://www.youtube.com/watch?v=3THrihH-oIM&t=410s
